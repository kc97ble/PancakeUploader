About Pancake Uploader


This is a project in Orbital Programme from NUS School of Computing


Motivation

This website is built with the purpose of providing an instant file sharing service.

While there are popular and widely used online storages like Google Drive, Microsoft OneDrive and Dropbox, these sites often require users to register for accounts and login before they can use the services. In many urgent situations when using file sharing service, users may feel inconvenient when they must type in account and password to login. Observing this fact, we decided to build a simple website for users to share their files without accounts!

Main Features

  • Upload/download files: Basic function for file sharing service
  • Shorten link for easy sharing: Users can share their files with ease as the link will be shorten.
